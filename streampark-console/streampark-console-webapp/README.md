## Local development

### Prerequisite

- [Node.js 16.0+](https://nodejs.org/en/download/)
- [pnpm](https://pnpm.io/installation)

### Build and run

1. Install dependencies:

```bash
pnpm install
```

2. Start a dev server:

```bash
pnpm serve
```

3. Build for production:

```bash
pnpm build
```
